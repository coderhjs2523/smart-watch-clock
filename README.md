# ⌚ Smart Watch Digital Clock

A modern Smart Watch inspired digital clock built using HTML, CSS, and JavaScript. The application displays the current time in real-time with a sleek and minimal user interface.

## 🚀 Features

- Real-time clock updates every second
- Smart watch inspired design
- Responsive layout
- Glowing visual effects
- Simple and lightweight implementation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Smart-Watch/
│
├── index.html
├── style.css
└── script.js
```

## 📸 Preview

Displays the current system time in a stylish smart watch interface.

## ⚙️ How It Works

- JavaScript fetches the current system time using the `Date()` object.
- The time is updated every second using `setInterval()`.
- CSS is used to create the smart watch appearance and glowing effects.

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/smart-watch-clock.git
```

2. Navigate to the project folder

```bash
cd smart-watch-clock
```

3. Open `index.html` in your browser.
